# No Hello

A minimalistic website explaining why saying just "hello" in chat is inefficient.

## About

This website educates people on better asynchronous communication practices. Instead of saying just "hello" and waiting for a response, include your actual question or context right away.

## Features

- Clean, professional design
- Interactive examples showing good vs. bad communication
- Fast loading with optimized performance
- Fully responsive for all devices
- Scroll animations and smooth UX

## Usage

Simply open `index.html` in a browser. No build process required - all dependencies are loaded via CDN.

## Tech Stack

- HTML5
- CSS3 (Tailwind CSS)
- Vanilla JavaScript
- Font Awesome icons
- Inter font
